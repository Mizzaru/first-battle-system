# First Battle System

My third Python project.

This project was my first attempt at building a simple turn-based battle system with health, mana, damage, healing, and offensive spells.

## Features

- Turn-based combat
- Health (HP) and Mana (MP) system
- Basic attack
- Healing spell
- Fireball spell
- Victory and defeat conditions
- Written entirely in Python

## What I learned

While working on this project I practiced:

- `while` loops
- Nested `if` statements
- Managing game state
- Updating player and enemy statistics
- Building simple game mechanics

## Why I made this project

After creating two text-based stories, I wanted to move beyond storytelling and start building actual gameplay mechanics.

This project became my first step into game programming.
